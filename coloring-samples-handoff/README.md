# Dark-mode "night" coloring twin samples (temporary handoff)

Review renders from the dark-twin experiment (direction B for dark mode — see PR #92
and ADR-0052). Generated by `scripts/gen-coloring-fills-dark.mjs`.

- `space/*.webp` + `nature/ladybug-tall.webp` — the generated night twins.
- `*.input.webp` — the inverted white-on-dark line art fed to Gemini.
- `space-contact.png` — a 6-page contact sheet of the Space category.

**This directory is temporary.** It lives here (not gitignored) only so a fresh session
can see the samples. Once you've reviewed them, **delete `coloring-samples-handoff/` and
let future generator runs write to the gitignored `.coloring-samples-dark/`** (the
script's default output dir) instead — don't keep committing sample binaries to the repo.
